# 溶液系统配置工具

## 项目概述

这是一个用于工业溶液系统设备配置的Web工具，帮助工程师快速完成多物料、多模块的设备参数配置。

**核心价值**：将复杂的Excel配置表转化为结构化的Web表单，支持批量操作、多物料管理、配置对比等功能，大幅提升配置效率。

---

## 主要功能

### 1. 多物料管理
- 支持多个物料标签页，每个物料独立配置
- 物料可拖拽排序、双击改名、删除
- 实时显示每个物料的配置完成度

### 2. 配置表单
- **简单选择**：下拉选择或手动输入
- **复合字段**：多个子字段组合，自动匹配型号
- **多选模式**：支持勾选多个选项
- 每个配置项支持备注

### 3. 配置表模板
- 从Excel自动识别配置结构（模块→配置项→选项）
- 支持两种格式：
  - `型号:字段1+字段2+...`（复合字段）
  - `字段1，字段2，字段3`（逗号分隔）
- 模板保存在localStorage，下次打开自动加载

### 4. 导入导出
- **导入Excel**：从已有配置表导入数据
- **导出Excel**：生成带格式的Excel文件（含边框、合并单元格）

---

## 新增功能（增强版）

### 5. 撤销/重做
- 支持50步操作历史
- 快捷键：`Ctrl+Z` 撤销，`Ctrl+Y` 重做
- 工具栏按钮：↩ 撤销，↪ 重做

### 6. 配置对比
- 选择两个物料进行对比
- 差异项高亮显示：
  - 红色：基准有/目标无
  - 绿色：基准无/目标有
  - 黄色：值不同

### 7. 批量修改
- 选择多个物料作为目标
- 选择模块→配置项→值
- 一键应用到所有选中物料

### 8. 项目存档系统
- 按项目名称/项目号保存配置
- 存档列表：查看、加载、删除历史项目
- **导出/导入存档**：保存为JSON文件，可备份或迁移

## 技术栈

- **前端**：纯HTML/CSS/JavaScript（无框架依赖）
- **Excel处理**：xlsx-js-style（SheetJS分支，支持样式）
- **存储**：localStorage（浏览器本地存储）
- **兼容性**：Chrome、Firefox、Edge现代浏览器

## 使用流程

### 首次使用
1. 打开 `溶液系统配置工具-增强版.html`
2. 点击「📋 配置表」导入Excel配置表模板
3. 系统自动识别模块和配置项结构
4. 开始逐个物料配置

### 日常使用
1. 选择物料标签页
2. 填写各配置项（下拉选择或手动输入）
3. 使用「多选」按钮开启多选模式
4. 配置完成后点击「📤 导出」生成Excel

### 存档管理
1. 点击「📁 存档」打开存档面板
2. 输入项目名称，点击保存
3. 历史存档可随时加载或删除
4. 使用「📤 导出全部存档」备份到文件

## 注意事项

1. **数据存储**：配置数据保存在浏览器localStorage中，清除浏览器数据会丢失
2. **建议存档**：重要配置请及时存档并导出备份
3. **配置表格式**：导入的Excel需要符合特定格式，详见配置表示例
4. **浏览器兼容**：推荐使用Chrome，旧版IE不支持
